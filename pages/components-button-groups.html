Button Groups

Group a series of buttons together on a single line or stack them in a vertical column.

  1. Dashboard
  2. Components
  3. Button Groups

Basic Example

Wrap a series of buttons with .btn in .btn-group. Add on optional JavaScript radio and checkbox style behavior with our buttons plugin.

HTML 
<div class="btn-group" role="group" aria-label="Basic example">
  <button type="button" class="btn btn-ocean">Left</button>
  <button type="button" class="btn btn-ocean">Middle</button>
  <button type="button" class="btn btn-ocean">Right</button>
</div>

Mixed Styles

You can mix different button styles within the same button group. Combine custom colors and outline styles for a unique look.

HTML 
<div class="btn-group" role="group" aria-label="Mixed styles example">
  <button type="button" class="btn btn-ocean">Ocean</button>
  <button type="button" class="btn btn-outline-emerald">Emerald</button>
  <button type="button" class="btn btn-rose">Rose</button>
  <button type="button" class="btn btn-outline-amber">Amber</button>
</div>

Button Toolbar

Combine sets of button groups into button toolbars for more complex components. Use utility classes as needed to space out groups, buttons, and more.

HTML 
<div class="btn-toolbar" role="toolbar" aria-label="Toolbar with button groups">
  <div class="btn-group me-2" role="group" aria-label="First group">
    <button type="button" class="btn btn-ocean">1</button>
    <button type="button" class="btn btn-ocean">2</button>
    <button type="button" class="btn btn-ocean">3</button>
    <button type="button" class="btn btn-ocean">4</button>
  </div>
  <div class="btn-group me-2" role="group" aria-label="Second group">
    <button type="button" class="btn btn-emerald">5</button>
    <button type="button" class="btn btn-emerald">6</button>
    <button type="button" class="btn btn-emerald">7</button>
  </div>
  <div class="btn-group" role="group" aria-label="Third group">
    <button type="button" class="btn btn-rose">8</button>
  </div>
</div>

Sizing

Instead of applying button sizing classes to every button in a group, just add .btn-group-lg, .btn-group-sm to each .btn-group, including each one when nesting multiple groups.

HTML 
<div class="btn-group btn-group-lg" role="group">
  <button type="button" class="btn btn-ocean">Left</button>
  <button type="button" class="btn btn-ocean">Middle</button>
  <button type="button" class="btn btn-ocean">Right</button>
</div>

<div class="btn-group" role="group">
  <button type="button" class="btn btn-ocean">Left</button>
  <button type="button" class="btn btn-ocean">Middle</button>
  <button type="button" class="btn btn-ocean">Right</button>
</div>

<div class="btn-group btn-group-sm" role="group">
  <button type="button" class="btn btn-ocean">Left</button>
  <button type="button" class="btn btn-ocean">Middle</button>
  <button type="button" class="btn btn-ocean">Right</button>
</div>

Vertical Variation

Make a set of buttons appear vertically stacked rather than horizontally. Split button dropdowns are not supported here.

HTML 
<div class="btn-group-vertical" role="group" aria-label="Vertical button group">
  <button type="button" class="btn btn-ocean">Button</button>
  <button type="button" class="btn btn-ocean">Button</button>
  <button type="button" class="btn btn-ocean">Button</button>
  <button type="button" class="btn btn-ocean">Button</button>
  <button type="button" class="btn btn-ocean">Button</button>
</div>

Button Groups with Icons

Add icons to button groups for better visual representation and context.

HTML 
<div class="btn-group" role="group">
  <button type="button" class="btn btn-ocean">
    <i class="bi bi-arrow-left me-1"></i>Previous
  </button>
  <button type="button" class="btn btn-ocean">
    <i class="bi bi-house me-1"></i>Home
  </button>
  <button type="button" class="btn btn-ocean">
    Next<i class="bi bi-arrow-right ms-1"></i>
  </button>
</div>

Nesting

Place a .btn-group within another .btn-group when you want dropdown menus mixed with a series of buttons.

  • Dropdown link
  • Dropdown link
HTML 
<div class="btn-group" role="group">
  <button type="button" class="btn btn-ocean">1</button>
  <button type="button" class="btn btn-ocean">2</button>
  <div class="btn-group" role="group">
    <button type="button" class="btn btn-ocean dropdown-toggle" data-bs-toggle="dropdown">
      Dropdown
    </button>
    <ul class="dropdown-menu">
      <li><a class="dropdown-item" href="#">Dropdown link</a></li>
      <li><a class="dropdown-item" href="#">Dropdown link</a></li>
    </ul>
  </div>
</div>

Radio Button Groups

Combine button-like radio and checkbox into seamless looking button groups. Add data-bs-toggle="buttons" to a .btn-group containing those modified buttons to enable their toggled state via JavaScript.

HTML 
<div class="btn-group" role="group">
  <input type="radio" class="btn-check" name="options" id="option1" autocomplete="off" checked>
  <label class="btn btn-outline-ocean" for="option1">Radio 1</label>

  <input type="radio" class="btn-check" name="options" id="option2" autocomplete="off">
  <label class="btn btn-outline-ocean" for="option2">Radio 2</label>

  <input type="radio" class="btn-check" name="options" id="option3" autocomplete="off">
  <label class="btn btn-outline-ocean" for="option3">Radio 3</label>
</div>

Checkbox Button Groups

Button groups can also function as radio button groups or checkbox button groups. Add data-bs-toggle="buttons" to enable the toggling functionality.

HTML 
<div class="btn-group" role="group">
  <input type="checkbox" class="btn-check" id="btncheck1" autocomplete="off">
  <label class="btn btn-outline-emerald" for="btncheck1">Checkbox 1</label>

  <input type="checkbox" class="btn-check" id="btncheck2" autocomplete="off">
  <label class="btn btn-outline-emerald" for="btncheck2">Checkbox 2</label>

  <input type="checkbox" class="btn-check" id="btncheck3" autocomplete="off">
  <label class="btn btn-outline-emerald" for="btncheck3">Checkbox 3</label>
</div>
