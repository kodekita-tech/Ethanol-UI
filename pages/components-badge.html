Pill badges

Use the .rounded-pill utility class to make badges more rounded (with a larger border-radius and additional horizontal padding). Combine with custom badge color classes for a modern look.

Positioned

Use utilities to position badges within buttons or other components.

HTML 
<button type="button" class="btn btn-primary position-relative">
  Inbox
  <span class="position-absolute top-0 start-100 translate-middle badge rounded-pill badge-rose">
    99+
    <span class="visually-hidden">unread messages</span>
  </span>
</button>

Badge in Buttons

Badges can be used as part of links or buttons to provide a counter.

HTML 
<button type="button" class="btn btn-primary">
  Notifications
  <span class="badge badge-slate ms-2">4</span>
</button>
