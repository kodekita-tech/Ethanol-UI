Button

Documentation and examples for buttons, our versatile component for actions and links.

  1. Dashboard
  2. Components
  3. Button

Example

Buttons are built with custom gradient backgrounds and modern styling. Use our button classes for consistent, beautiful buttons throughout your application.

HTML 
<button type="button" class="btn btn-ocean">Ocean</button>
<button type="button" class="btn btn-slate">Slate</button>
<button type="button" class="btn btn-emerald">Emerald</button>
<button type="button" class="btn btn-rose">Rose</button>
<button type="button" class="btn btn-amber">Amber</button>
<button type="button" class="btn btn-cyan">Cyan</button>
<button type="button" class="btn btn-stone">Stone</button>
<button type="button" class="btn btn-violet">Violet</button>
<button type="button" class="btn btn-gradient">Gradient</button>

Button Sizes

Fancy larger or smaller buttons? Add .btn-lg, .btn-sm for additional sizes.

HTML 
<button type="button" class="btn btn-ocean btn-lg">Large</button>
<button type="button" class="btn btn-ocean">Default</button>
<button type="button" class="btn btn-ocean btn-sm">Small</button>

Buttons with Icons

Add icons to buttons for better visual representation and context.

HTML 
<button type="button" class="btn btn-emerald">
  <i class="bi bi-check-circle me-2"></i>Success
</button>
<button type="button" class="btn btn-rose">
  <i class="bi bi-x-circle me-2"></i>Error
</button>

Outline Buttons

Need an outlined button style? Add .btn-outline-* classes for a button without background.

HTML 
<button type="button" class="btn btn-outline-ocean">Ocean</button>
<button type="button" class="btn btn-outline-slate">Slate</button>
<button type="button" class="btn btn-outline-emerald">Emerald</button>
<button type="button" class="btn btn-outline-rose">Rose</button>

Disabled State

Make buttons look inactive by adding the disabled boolean attribute to any button element.

HTML 
<button type="button" class="btn btn-ocean" disabled>Disabled</button>
<button type="button" class="btn btn-emerald" disabled>Disabled</button>
<button type="button" class="btn btn-outline-rose" disabled>Disabled</button>

Loading State

Show loading state by adding a spinner to your button.

HTML 
<button type="button" class="btn btn-ocean" disabled>
  <span class="spinner-border spinner-border-sm me-2" role="status" aria-hidden="true"></span>
  Loading...
</button>
