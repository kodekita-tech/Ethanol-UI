Breadcrumb

Documentation and examples for breadcrumb navigation, indicating the current page's location within a navigational hierarchy.

  1. Dashboard
  2. Components
  3. Breadcrumb

Basic Example

A simple breadcrumb navigation component to show the current page's location in a hierarchical structure.

  1. Home
  2. Library
  3. Data
HTML 
<nav aria-label="breadcrumb">
  <ol class="breadcrumb">
    <li class="breadcrumb-item">
      <a href="#">Home</a>
    </li>
    <li class="breadcrumb-item">
      <a href="#">Library</a>
    </li>
    <li class="breadcrumb-item active" aria-current="page">
      Data
    </li>
  </ol>
</nav>

Breadcrumb Divider

Change the divider by modifying the CSS custom property --bs-breadcrumb-divider or using the $breadcrumb-divider Sass variable.

  1. Home
  2. Library
  3. Data
  1. Home
  2. Library
  3. Data
  1. Home
  2. Library
  3. Data
  1. Home
  2. Library
  3. Data
HTML 
<!-- With > divider -->
<nav aria-label="breadcrumb">
  <ol class="breadcrumb" style="--bs-breadcrumb-divider: '>'">
    <li class="breadcrumb-item"><a href="#">Home</a></li>
    <li class="breadcrumb-item"><a href="#">Library</a></li>
    <li class="breadcrumb-item active" aria-current="page">Data</li>
  </ol>
</nav>

<!-- With | divider -->
<nav aria-label="breadcrumb">
  <ol class="breadcrumb" style="--bs-breadcrumb-divider: '|'">
    ...
  </ol>
</nav>

<!-- With → divider -->
<nav aria-label="breadcrumb">
  <ol class="breadcrumb" style="--bs-breadcrumb-divider: '→'">
    ...
  </ol>
</nav>

Product Details

This is an example of breadcrumb navigation inside a card component.

HTML 
<div class="card">
  <div class="card-header bg-transparent border-0 pb-0">
    <nav aria-label="breadcrumb">
      <ol class="breadcrumb mb-0">
        <li class="breadcrumb-item">
          <a href="#"><i class="bi bi-house me-1"></i>Home</a>
        </li>
        <li class="breadcrumb-item">
          <a href="#">Products</a>
        </li>
        <li class="breadcrumb-item active" aria-current="page">
          Product Details
        </li>
      </ol>
    </nav>
  </div>
  <div class="card-body">
    <h5 class="card-title">Product Details</h5>
    <p class="card-text">...</p>
  </div>
</div>
